# 第一条视频生产流程记录 v1

目标：跑通一条带配音的横屏小红书教程视频。

主题：我用 AI 跑通了一个小红书视频生成器。

## 已验证闭环

1. ChatGPT 负责：
   - 定选题
   - 写口播稿
   - 拆分镜
   - 写 Remotion 模板代码
   - 更新 JSON 配置
   - 给 Codex 最小执行指令

2. GitHub 负责：
   - 保存项目代码
   - 保存模板和配置
   - 记录每次修改

3. Codex 负责：
   - 本地拉取代码
   - 安装依赖
   - 类型检查
   - 复制本地素材
   - 渲染视频

4. Remotion 负责：
   - 读取 JSON
   - 读取截图和配音
   - 渲染 MP4

## 第一条视频素材

本地素材位置：

```text
public/episodes/001/assets/
```

需要的文件：

```text
01-github.png

```

注意：真人配音不提交 GitHub，只放本地。

## 已跑通输出

```text
out/landscape-episode-001.mp4
```

## 当前能力

- 支持横屏教程视频
- 支持截图素材
- 支持配音音频
- 支持底部字幕条
- 支持章节标题
- 支持右上角提示气泡
- 支持进度条

## 当前限制

- 还不能自动识别音频
- 还不能自动生成字幕时间轴
- 还不能自动匹配素材
- 还不能一键生成完整视频

## 下一步升级方向

1. 主题和口播稿自动拆成 JSON 分镜。
2. 支持每期独立 episode 文件。
3. 支持录屏素材。
4. 支持音频转字幕。
5. 根据字幕自动匹配截图或卡片。
